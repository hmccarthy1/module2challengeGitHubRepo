About Me

Learning Path

Excel vba

Excel VBA is a tool to quickly manipulate data in excel in an automated fashion that would be tedious to do manually. I learned this working in Walmart's supply chain in a position that required a lot of data analytics.

Microsoft Power FX

Power FX is the language behind Microsoft's Power Platform, which is Microsoft's solution for low-code rapid software development. I self-taught, learning the ins and outs of provisioning data tables, managing data relationships, UI/UX, etc.

HTML, CSS, JS

I learned these coding languages as part of UCF's coding bootcamp, which is a high-velocity, high-intensity coding bootcamp designed to bring participants up to the skill level needed to become a full-stack web developer. I learned all about these languages, the software development lifecycle, and developing software as part of a team.
